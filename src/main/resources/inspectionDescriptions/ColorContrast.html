Color contrast inspection is an automated inspection to check in xml layout files if color contrast between background and textcolor respects the WCAG 2.0 standard of 4.5:1 for non-bold, under 18pt text.

Checks automatically is the colors used in the attributes "android:background" and "android:textColor" respect the WCAG standard, works with hard coded color hex or with references (@colors/xxx).

See https://jetbrains.design/intellij/text/inspections/#descriptions for more information.
